- コンセプト：倒した敵に乗り移って殲滅していくFPS
- 制作期間：10ヶ月（2024年4月～2025年1月）
- チーム構成：プランナー2人、プログラマー4人（本人含む）、デザイナー2人
- 開発環境：Unity(2021.3.25f1)/Visual Studio 2022
- 担当箇所：
